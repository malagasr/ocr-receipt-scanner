Features

Everything you need to
track spending effortlessly

From raw receipt image to structured spending data in seconds. No manual entry required.

📸

Multi-format upload

Drag & drop JPG, PNG, or PDF receipts. Batch process multiple files at once with real-time progress tracking.

🔍

Triple OCR engines

Tesseract, EasyOCR, and Vision API work together. Automatically picks best result for maximum accuracy.

🤖

AI parsing & categorization

Ollama LLM extracts store name, date, total, and every line item. Automatically categorizes by spend type.

📊

Interactive analytics

Plotly charts show spending trends over time, category breakdowns, and merchant frequency maps.

🗄️

Local SQLite database

All data stays on your machine. No cloud, no subscriptions, no privacy concerns. Export to CSV or JSON anytime.

⚡

Real-time status

Live system health panel shows OCR engine status, database connections, and processing queues at a glance.

How it works

From photo to insight
in four steps

The pipeline is fully automated. You upload — the app handles the rest.

01

Upload receipt

Drop an image or PDF. Any angle, any lighting. The preprocessor normalizes it before OCR.

02

OCR extraction

Three engines run in parallel. The highest-confidence result moves forward to the AI parser.

03

AI parsing

LLM structures raw text into merchant, date, total, line items, and spending category.

04

Dashboard & export

Data lands in SQLite instantly. Charts update live. Export whenever you want.
